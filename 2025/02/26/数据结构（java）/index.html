ArrayList大小没有显示，可以添加或者删除元素，相当于python的list

注意ArrayList隶属于集合，一般存储的是对象，所以不能用基本类型int，需要用int的包装类型Integer

  • remove（index）：删除指定位置的元素
  • set（index, val）：修改index位置的元素为val

StringBuilder

对字符串进行修改的时候用的，更多的是可以直接当字符串的数组来用

StringBuilder比StringBuffer快，对字符串进行修改推荐使用StringBuffer

  • append(str): 添加string
  • insert(pos, str): 在指定pos处开始，插入str
  • replace(idx1, idx2, str)：从idx1到idx2处替换为str
  • delete(idx1, idx2): 删除idx1到idx2
  • reverse(): 反转
  • int = capacity() / int = length() : 长度
  • toString(): 转化为string，一般用于输出
  • setLength(int): 相当于python的 s[-k:]，取后k个

接下来的都是使用LinkedList实现的，是一个顺序容器，它的底层是通过双向链表实现的

栈

这里有好几种说法，首先是使用stack类，但是这个是java早期的实现，现在已经呗标记为遗留，不推荐

第二个是跟python一样，可以使用Deque接口来实现栈，只是自己保证FIFO就行

常用的两个实现类是ArrayDeque, LinkedList

这里的操作建议还是使用栈的操作(左边)，而不是队列的(右边)，但是知道对应关系就行

  • push() / addFirst(): 压栈

  • pop() / removeFirst(): 出栈

  • peek() / peekFirst(): 读取栈顶

  • addAll(collection): 集合添加元素，相当于python的extend，注意只能添加的是集合类型，一般用多的是List直接添加到栈中

  • descendingIterator: 迭代器，从栈顶到栈底，用于按出栈顺序读取栈的元素，.next()方法去下一个元素,使用方法：

队列

可以使用Queue接口，也可以使用Deque接口，实现的类都是LinkedList

  • offer():添加元素
  • poll(): 返回第一个元素，删除
  • peek() / element()：返回第一个元素

我觉得更通用的是deque接口：

记住队列是队首进，队尾出，栈是队首进，队首出

  • addLast() / offerLast(): 添加元素
  • removeFirst() / pollFirst(): 返回第一个元素，删除
  • getFirst() / peekFirst(): 返回第一个元素

两种实现的区别：

使用Deque，推荐总是明确调用offerLast()/offerFirst()或者pollFirst()/pollLast()方法。

  • poll() : 出堆
