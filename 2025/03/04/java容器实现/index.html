1.初始化

默认大小16，负载因子为0.75，当元素达到16 * 0.75 = 12之后就触发扩容

• JDK 1.7 及以前：采用 链地址法（拉链法），即所有冲突的元素形成一个 单向链表。

• JDK 1.8 及以后：

​ • 当链表长度 ≤ 8：仍然使用 链表 结构。

​ • 当链表长度 > 8 且桶内元素 ≥ 64：转为 红黑树 结构，提高查询效率（O(logN)）。

​ • 当链表长度 ≤ 6：重新转换为 链表。
