8.1.2如何更新某个节点

改变某个index对应的值，就只需要去找包含该index的关键区间就行，怎么去找这个关键区间？

1.右端点为x的区间，包含且仅包含于右端点为x + lowbit(x)的区间

更新的算法如下：

假设index的节点变为val：

  • right = index + 1(right对应的是tree中的索引，是从1开始的，index是nums中对应的索引，从0开始)
  • 需要更改的区间为tree[right]，所以tree[right] += (val-index)
  • right = right + lowbit(right)，循环更新right之后更新tree[right]，直到i > len(nums)

8.1.3如何计算区间的合

假如我要计算[left, right]的区间合

  • 先找到[1, right + 1] , [1, left]
  • [1, left] = tree[left], [1,right + 1] = tree[right + 1]
