python内置函数

发表于 2024-05-29 更新于 2025-01-06 分类于 python

主要是为了记录一下刷题用到的一些python的内置函数

lambda表达式

（匿名函数）

lambda arguments: expression

arguments：参数列表，在冒号前指定

expression :表达式，用于计算并返回函数的结果

reduce()

python3需要导入functools

累积函数，对一个数据集合中的所有数据进行操作，主要是配合lambda使用

enumerate（）

用于将一个可遍历的数据对象(如列表、元组或字符串)组合为一个索引序列，同时列出数据和数据下标，一般用在 for 循环当中

zip()

用于将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的列表

假如两个的长度不想等，等于最小的长度

accumulate()

其实这个并不是属于自带的内置函数，是在itertools里面的，但是还是这里写

主要功能：对可迭代的对象进行操作（可以是累加，累乘或者别的func）

用法：

accumulate(iterable[,func , * , initial = None])

iterable : 可迭代的对象

func : 带两个参数的函数

initial ： 开始的值

但是得到的是一个对象，可以强制转化为list都行

.sort()

用于list排序的函数：

list.sort(cmp=None, key=None, reverse=False)

cmp: 可选参数，指定了就使用该参数的方法进行排序

key : 用于比较多元素，指定可迭代对象中的一个元素来进行排序

reverse : 排序规则，True 降序 ， False 升序

指定元素：

filter()

用于过滤序列，返回一个迭代器对象，可以使用list()来转换

filter(function, iterable)

function 判断函数

all()

用于判断可迭代参数中所有元素是否都为True,如果都为True返回值为True，否则返回值为False

一般在算法题中的用法，可以验证某个可迭代的容器中元素，是否都满足某个条件

split（）

按指定分隔符对字符串进行切片，并返回一个列表

str 分隔符，默认为空字符

num 分割的次数，如果为-1就是分割所有

这是python内置的分割函数，只能支持单个分割符

如果要多个，使用re.split

re.split()

同样的用法，用|隔开分割符，这个主要是跟正则表达式联系起来使用的，得到的是一个字符串列表

str.isdigit()

检测字符串是否只由数字组成，只对 0 和 正数有效，如果字符串只包含数字则返回 True 否则返回 False。

format()方法

用于格式化string，在字符串中使用{}表示需要插入的位置

pairwise()

属于itertools库，是一个迭代器，从对象中获取连续的重叠对

str.join()

在 Python 中，.join() 方法是一个字符串方法，用于将一个可迭代对象（如列表、元组等）中的元素连接成一个字符串，并且在这些元素之间插入指定的分隔符。

需要注意的是，.join() 方法只能用于包含字符串元素的可迭代对象。如果可迭代对象中包含非字符串类型的元素，则需要先将它们转换为字符串。
