Vim学习笔记

发表于 2024-06-01 更新于 2024-08-27 分类于 工具使用

学习的资料是：《Vim实用技巧》，

先使用自带的vimtutor进行上手

基础操作

这些基础操作可以简单的上手vim，如果需要更近一步，需要看之后的部分

h 左， （在最左边）

l 右， （在最右边）

j 下，（看起来像一个向下的箭头）

k 上

w 下一个字开头

e 下一个字结尾

b 上一个字的开头

gg / shift + g 文章开头 / 文章末尾

:q! 直接退出，不保存

:wq 保存并退出

x 删除当前字符

shift + x 从后往前删除光标所在位置

dd 删除当前行

i 插入文本，当前位置进入插入模式

o 下一行插入文本

a 当前位置插入

A 添加内容

yy 复制当前行

p 粘 ， 小写在光标位置之后，大写在之前

大范围移动操作

[number] 向下移动多少行（最常用）

[number] + [space] 输入一个数字，再按空格，光标向右移动number个字符

e 移动到下一个单词的末尾（end）

w 移动到下一个单词的开头

b 移动到上一个单词的开头

ge 移动到上一个单词的末尾ljjlll

ctrl + f 下一页，相当于page down

ctrl + b 上一页，相当于page up

ctrl + d 向下移动半页

ctrl + u 向上移动半页

H 移动到屏幕最上面行的第一个字符

M 移动到屏幕最中央行的第一个字符

L 移动到屏幕最下面行的第一个字符

G 移动到文档的最后一行

[number] + G 移动到文档的第number行(常用)

0 跳转到当前行的最前面

删除类指令

许多的命令都是由一个操作符和一个动作构成，删除的命令格式如下：

x 删除光标处的单词

dw 删除单词（delete word）

de 删除当前位置到单词末尾，包括最后一个字符，dw只删除单词，de删除到下一个单词，假如两个单词之间有多个空格啥的，de全删，dw不删

d$ 从光标处删除到当前行尾部

在动作前面输入数字会使他重复那么多次
可以通过输入计数器的方式，删除多个

dd 删除整行，同理可以使用计数器删除多行：2dd

u 可以撤销最后的指令 （ctrl+r）

U 可以撤销整行的指令

置入类指令

与删除类指令相同，置换类也可以用命令加操作符完成：

动作参数(motion)也是一样的，比如 w 代表单词，$代表行末等等

y 复制所选内容

yy 复制当前行

p 粘贴（Paste），使用dd删除之后，内容会保存到vim的寄存器里面（感觉相当于windows的剪切板），然后可以使用p指令将寄存器中的内容粘贴

r 替换（repalce），输入r和一个字符，可以替换光标所在位置的字符

d 剪切

ce改变文本直到一个单词的末尾（change end）， 相当于改变

一共11个粘贴板，分别是0 ~ 9， a, "，如果开启系统剪贴板，有两个是+ , *。 使用 :reg可以查看各个剪贴板里面的内容 ，默认使用的y, p都是从"剪贴板中进行复制粘贴

粘贴板号 + y/p可以指定粘贴板复制粘贴（正常模式）

位置类指令

Ctrl + g可以查看当前行的信息，可以看目前在全文的多少行

[number] + G 跳转到多少行

G文件最后一行

gg文件第一行

/ + [字符串] 可以查找当前文件中查找该字符串(往后)

？ + [字符串] 可以向前查找文件中的该字符串(往前)

n （next）与指令相同方向下一个

N （Next）与指令相反方向下一个

Ctrl + o 回退

% 把光标放到括号的位置，按%后跳到匹配的括号的位置，再按一次回到配对的第一个括号的地方

:s/old/new/g 可以替换 old 为 new(常用)

  • :#,#s/old/new/g 其中 #,# 代表的是替换操作的若干行中首尾两行的行号。
  • :%s/old/new/g替换整个文件中的每个匹配串
  • :%s/old/new/gc 会找到整个文件中的每个匹配串，并且对每个匹配串提示是否进行替换。
  • 这三个都挺常用的

命令

:![外部命令：如ls，cd这种] 执行外部的shell命令

:w [文件名] 将当前文件保存到文件名这个文件中

v + :w filename 可以只保存用v选中的内容到filename文件中

:r filename 提取filename的文件插入到当前文件中

o 在当前的下面新建一行，进入编辑模式

O（大写的o) 在上面新建一行，进入编辑模式

a 光标之后插入文本
